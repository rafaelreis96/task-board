✌ Task Board

Quadro para controle de tarefas

Task Board tem como propósito auxiliar usuários no acompanhamento e cumprimento de suas tarefas / metas.

Funcionalidades

  • ✓ Cadastro de tarefas
  • ✓ Consulta de tarefas
  • ✓ Lembrete para tarefas
  • ✓ Temporizador para tarefas
  • ✓ Quadro para acompanhamento facilitado
  • ✓ calculo de progresso / desempenho
